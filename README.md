# Webbase
## Webbase for free use

This webbase consists out of following technologies:
 - Web Framework: [express](https://www.npmjs.com/package/express)
 - View Engine: [ejs](https://www.npmjs.com/package/ejs)
 - ORM Framework: [sequelize](https://www.npmjs.com/package/sequelize)
 - Database: [sqlite3](https://www.npmjs.com/package/sqlite3) (configurable inside the sequelize loader)

Goal of this boilerplate was, to prepare a quick and easy to use tech-stack for my apprenticeship exam.
